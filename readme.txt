简介：这个工程构建了深圳公交网络与亳州公交网络。底层使用c语言实现，顶层使用python实现。依赖包包括：numpy+matplotlib+networkx+pyqt5

界面：界面里共有8个按钮，网络结构图按钮用于生成网络结构图（其中红点表示高度数结点（degree>=5），蓝点表示中等度数结点(2<=degree<5),黄色小节点表示低度数结点），度分布图按钮用于生成度分布图，刻意攻击按钮与随机攻击按钮分别表示刻意攻击与随机攻击，若攻击百分之十就点击一下，每多点击一下攻击节点数增加百分之十。攻击效果用于生成攻击效果图（包括了平均度数，平均聚簇系数，平均最大连通子图数）。最短路径按钮用于生成最短路径对比图（由于最短路径在大样本下生成速度很慢因此最调用最短路径函数生成后再传入gui界面），生成按钮用于生成网络，重置按钮用于重置网络。
目录： create_net.c是底层c语言的实现，gui.py是gui界面实现，gui.ui是gui的ui，libcreate_net是动态链接库。sz_bus是调用函数的example
数据：bus_data是深圳公交的地图 bus_data-bz是亳州公交数据。要使用自己的数据 只需要将名字改为bus_data即可。
